## 📖 About

AION2-TR is an open-source Turkish localization project created by the community.

Our goal is to provide high-quality Turkish translations for AION 2 while preserving the original game's atmosphere.

---

## ✨ Features

- Turkish UI
- Turkish Quests
- Turkish NPC Dialogues
- Turkish Item Descriptions
- Turkish Skill Descriptions
- Automatic Installer
- Automatic Updater

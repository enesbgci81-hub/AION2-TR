# AION2-TR Logo Specification

## Objective

Create a modern, minimal, and recognizable logo representing the Turkish localization project for AION 2.

---

## Design Goals

- Minimal
- Modern
- Gaming-oriented
- Easy to recognize
- Works on dark backgrounds
- Scalable (SVG)

---

Primary Blue

Turkish Red
#E30A17

Dark Background
#0F172A

White
#FFFFFF

## Typography

Recommended Fonts

- Inter
- Exo 2
- Orbitron (optional)

## Deliverables

- SVG Logo
- PNG (1024x1024)
- Transparent PNG
- Favicon
- GitHub Avatar
